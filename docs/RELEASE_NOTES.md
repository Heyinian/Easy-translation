## v0.3.0

发布时间：2026-03-21

### 本版本重点

- 输入窗口翻译机制完善
- 候选译文功能落地
- 本地设置与配置能力增强

### 用户可见变化

1. 输入窗口翻译支持 triple_space 和组合键两种模式
2. 三击空格窗口期可配置
3. 外部输入框翻译后可回填到原输入框
4. Ollama 翻译可展示多个候选结果
5. 可在界面中点击候选译文切换当前结果
6. 设置页支持更多运行时配置

### 修复内容

1. 修复翻译按钮点击时可能因信号参数类型导致的错误
2. 修复旧三击空格逻辑残留导致的误删输入问题
3. 修复输入窗口翻译触发方式与运行时状态不一致的问题

## v0.2.0

发布时间：2026-03-21

### 本版本重点

- 设置系统建立
- 多翻译源能力增强
- 智能翻译方向完善

### 用户可见变化

1. 新增设置对话框
2. 支持保存翻译源、热键、OCR、Tesseract 等配置
3. 支持百度和腾讯翻译
4. 默认模式支持智能中英方向判断
5. API 密钥改为本地加密存储
