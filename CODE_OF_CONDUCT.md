# 行为准则

## 我们的承诺

为了营造一个开放且热烈欢迎的环境，我们作为贡献者和维护者承诺参与我们的项目和社区的每个人都能享受到没有骚扰的体验，不管他们的年龄、身体特征、残疾、种族、性别、性别认同及表现、经验水平、学历、社会经济地位、国籍、个人相貌、种族、宗教或性取向如何。

## 我们的标准

有助于创造积极环境的行为包括：

- 使用欢迎和包容的措辞
- 尊重不同的观点和经验
- 优雅地表达批评
- 关注什么对社区最有利
- 对其他社区成员表现出同情

参与者不可接受的行为包括：

- 使用性意象和不受欢迎的性关注或骚扰
- 在线骚扰、恐吓或威胁
- 发表与个人的种族、性别、性别认同和表现相关的冒犯性评论
- 冒充骚扰或不尊重他人隐私
- 发布他人的私人信息，如地址或电子邮件
- 不适当的肢体接触及任何不受欢迎的性接触
- 会合理地被认为是在专业背景下不适当的其他行为

## 我们的职责

项目维护者负责澄清可接受行为的标准，并应对任何不可接受的行为采取适当的纠正措施。

项目维护者有权利和责任删除、编辑或拒绝评论、提交、代码、Wiki编辑、问题以及不符合本行为准则的其他贡献，或暂时禁止任何参与者。

## 范围

本行为准则适用于所有项目空间，也适用于代表项目或社区的个人在公共空间中的行为。代表项目或社区的例子包括使用官方项目电子邮件地址、通过官方社交媒体账户发布或在在线或离线活动中作为指定代表行动。

## 执行

可能的辱骂、骚扰、或其他不可接受的行为可被报告给管理员。所有投诉获得审议和调查，在必要时作出回应。
